TranscodeLib

A Library for transcoding any Media(Image/Video) as per requirement

Download this project as a .zip file Download this project as a tar.gz file

Project

Transcoding Library is a system which allows user to converting still frames into movies and converting a movie into image sequence that can be player back on a large screen or projector (and/or directly on artist's workstation) for the purpose of doing a review or check the resulting work in motion.
